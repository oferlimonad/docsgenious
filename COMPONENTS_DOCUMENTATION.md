# 📋 תיעוד מפורט של כל המרכיבים באתר DocsGenius

## 🎨 **קומפוננטות בסיסיות (UI Components)**

### 1. **Button** - כפתור
**מיקום בקוד:** שורה 78-102
**Variants (סוגים):**
- `primary` - כפתור ראשי (כחול-אינדיגו עם גרדיאנט)
- `secondary` - כפתור משני (שקוף עם גבול)
- `danger` - כפתור סכנה (אדום)
- `ghost` - כפתור שקוף (ללא רקע)
- `cta` - כפתור קריאה לפעולה (לבן)

**Sizes (גדלים):**
- `sm` - קטן
- `md` - בינוני (ברירת מחדל)
- `lg` - גדול
- `xl` - גדול מאוד

**תכונות:**
- תמיכה באייקון (`icon` prop)
- `disabled` state
- אפקט hover עם shadow

### 2. **Card** - כרטיס
